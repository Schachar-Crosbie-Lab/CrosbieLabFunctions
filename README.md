## Installation

You can install the development version of CrosbieLabFunctions from
[GitHub](https://github.com/) with:

``` r
### Use Pak
#install.packages('pak')
pak::pkg_install("jclutton/CrosbieLabFunctions")


### Use Devtools
# install.packages("devtools")
devtools::install_github('jclutton/CrosbieLabFunctions')
```
